Game guide
1. This game depend on destroying ships by shoting to them.
2. Player have at one's disponal board in which located random drawed ships. Every ship is hidden and player have to trying find him by shooting to him.
3. To shoot towards ship player have to choose correct ship coordination and click "FIRE" button.
4. Ship coordination looks as follows: B7
If player write other value than value which includes big letter ahead and number at the back, game return statement about uncorrected ship coordination.
5. Game is finish once player destroyed all ships. Information about currently destroyed ships located on bottom game part.
6. Player can repeat every game. In new game ships spring out in new places.
